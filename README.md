## Project Description

This project is a **Student Course Registration Management System** built with **Node.js** and **Express.js**, using **EJS** for server-side rendering and **PostgreSQL** as the database, following an **MVC architecture**.

It provides features to manage **students**, **subjects**, and **course registrations** (including score updates), with full **CRUD operations** (Create, Read, Update, Delete). The application also includes **search**, **pagination**, and **session-based success/error messages** to improve usability.

### Technologies Used

- **Node.js** (runtime environment)
- **Express.js** (web framework)
- **EJS** + **express-ejs-layouts** (template engine and layout management)
- **PostgreSQL** + **pg** (database and Node.js PostgreSQL client)
- **express-session** (session handling for notifications/messages)
- **dotenv** (environment variable management)
- **HTML/CSS/Bootstrap** (UI styling in server-rendered views)
- **JavaScript** (frontend interactions and backend logic)
